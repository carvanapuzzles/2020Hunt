The puzzle has 6 clues but 12 answer spots. This, along with "Crossover" in the title should hint that there are two answers per clue. The clues hint at video games and each clue can be answered from the perspective of either Minecraft or Animal Crossing: New Horizons. The answers are (Minecraft/ACNH):

Currency used to buy items in-game (singular) - EMERALD / BELL
Game developer company - MOJANG STUDIOS / NINTENDO
Object you use to craft new tools - CRAFTING TABLE / DIY WORKBENCH
Plant that grows in sand - CACTUS / COCONUT TREE
Very common ocean fish - TROPICAL / SEA BASS
You need an odd number of this item to craft the most basic fishing rod (singular) - STICK / TREE BRANCH

The answers can be arranged in the grid with ACNH answers running vertically and Minecraft answers running horizontally.
